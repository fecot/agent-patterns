# ブログ記事用の素材

このディレクトリは、**後で Claude Code にブログ記事を書かせるための入力素材**です。実装と同時に、Phase ごとの開発ログを `devlog/` に書き溜めています。

## 使い方（記事を書かせるときの指示テンプレート）

> `docs/blog/devlog/0X-phaseX.md` と `docs/blog/article-template.md` を読んで、Phase X の実装を解説するブログ記事を書いてください。
> 制約:
> - 業界固有語（PR・広報・メディア 等）は使わない
> - 対象読者は「社内業務エージェントをこれから作るエンジニア」
> - devlog の「ハマったポイント / 設計判断」を記事の山場にする
> - 試せるコマンドと期待出力を必ず含める

## 構成

- `article-template.md` … 記事の骨子テンプレート
- `devlog/00-phase0.md` 〜 `03-phase3.md` … Phase ごとの開発ログ（記事の素材）

## ルール

- 業界固有語を使わない（素材 MD にも適用）。
- 各 Phase 完了時に対応する devlog を更新してからコミットする（実装とログをずらさない）。
