# Report Writing Guide

Report（レポート）作成のガイドです。Report Assistant が集計・要約を行う際の指針にもなります。

## 構成

1. 対象期間と集計観点（department / category など）
2. 主要な指標（件数、status 別内訳、priority 別内訳）
3. 傾向の要約（増減と、その背景の仮説）
4. 注目すべき Record / Case

## 守ること

- 集計の数値は入力データに基づきます。**データにない数値を作らない（捏造しない）**。
- 仮説と事実を区別して書きます。
- 結論には根拠となる指標を添えます。

## 出力形式

初期は docx を生成し、MinIO に保存してダウンロード URL を発行します。
