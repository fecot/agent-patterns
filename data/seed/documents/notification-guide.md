# Notification Guide

Notification（通知）の下書きと送信に関するガイドです。

## 基本方針

- Assistant は Notification の **下書き（draft）** までを作成します。
- 実際の送信は人間が内容を確認してから行います（自動送信はしません）。

## 下書きに含める要素

1. 宛先の役割（role）
2. 用件の要約（1〜2 文）
3. 必要なアクションと期限
4. 関連する Record / Case への参照

## トーン

- 簡潔で中立的に書きます。
- 断定が必要な箇所には根拠（Record や Document）を添えます。
